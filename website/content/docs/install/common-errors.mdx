- CSL0000: BasicError
  Fix it yourself
- CSL0100: ACL not found
  There is not an ACL with this ID. Check that you have the correct ACL ID or create an ACL with this ID if needed.
- CSL0101: Cannot resolve root ACL
  Use a non-root ACL token
- CSL0102: ACL support disabled
  ACL changes are not permitted since they are disabled. Enable ACLs via the -enable-acl command line option or the enable_acl configuration option.
- CSL0103: Permission denied
  Check the ACL rules
- CSL0104: Invalid Parent
  Check the ACL rules
